# Screenshots

Use screenshots when visual layout, styling, images, or exact page appearance matters. Prefer `safari_snapshot` for normal interaction because it is structured and cheaper.

If screenshots fail:
- Run `safari_setup_status`.
- Check Screen Recording permission for Codex.
- If the extension is disconnected, the plugin may use AppleScript or native fallback paths depending on the page and permissions.

Do not include screenshots containing sensitive information in external messages unless the user explicitly asks to share them.
